## 🎨 Design System Overview

### Colors (Premium Dark Void)
```
Background:    #09090b (zinc-950)
Text Primary:  #f4f4f5 (zinc-100)
Text Secondary: #a1a1aa (zinc-400)
Accent Primary: #06b6d4 (cyan-500)
Accent Sec:    #3b82f6 (blue-500)
```

### Typography
- **Display**: Space Grotesk (headings)
- **Body**: Geist Sans (body text)
- Both use system font stack fallbacks

### Visual Effects
- Noise overlay (3% opacity on desktop, 1.5% mobile)
- Spotlights (cyan/blue, responsive blur)
- Glass containers (backdrop-blur-md)
- Text gradients (cyan→blue)
- Thin borders (border-white/5)

### Responsive Scaling
- **Mobile (<640px)**: 30% lighter graphics, reduced blur
- **Tablet (641-1024px)**: Medium optimization
- **Desktop (1025px+)**: Full effects enabled
