# Tech Stack & Modern Libraries

This renovation leverages standard React rendering and advanced animation packages optimized for React 19 and Next.js 16.

## Core Stack
- **Next.js 16 (App Router)**: Utilizing Next's native font optimization (`next/font`), search forms (`next/form`), and SSR/CSR boundaries.
- **React 19**: Leverages standard hooks, compiler caching, and high-fidelity rendering.
- **Tailwind CSS v4**: CSS-first configuration, modernized class syntax, custom @theme properties.

## Animation & 3D Foundations
- **Smooth Scrolling (Lenis)**: Powered by `@studio-freight/react-lenis` to deliver unified scroll-friction mechanics across all form factors.
- **GSAP + ScrollTrigger**: Dedicated engine for complex scroll-driven timeline orchestrations.
- **Framer Motion**: State-driven visual changes, exit animations, hover scaling, drag constraints, and physics-based interactions.
- **React Three Fiber (Three.js)**: Canvas rendering ecosystem for the Hero interactive spatial animations.

## Component Libraries
- **Shadcn UI**: Accessible, well-structured structural blocks (Sheet, Dialog, Button, Carousel).
- **Aceternity UI / Magic UI**: Pre-engineered high-impact motion elements:
  - Sparkles / Aurora background glows.
  - 3D Pin Cards (interactive isometric mouse-tilting).
  - Tracing Beams (scroll progress tracking path).
  - Infinite Marquees (for continuous skill stack rendering).
