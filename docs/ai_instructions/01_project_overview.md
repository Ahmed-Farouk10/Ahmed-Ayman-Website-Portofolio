# Project Overview: Portfolio Renovation

## The Goal
To renovate an existing Next.js 16 + React 19 + Tailwind CSS v4 developer portfolio into a pioneering, visually stunning, and highly interactive developer workspace. The core strategy is to elevate frontend aesthetics and creative motion to absolute premium status, keeping the backend data structure in Sanity completely untouched.

## Strict Boundaries
- **DO NOT** modify anything inside the `/studio` directory. The Sanity backend schemas and configurations must remain intact.
- **Frontend Focus**: All work is confined to the `/web` directory (the Next.js frontend).
- **Data Preservation**: Do not remove, destroy, or break the existing Sanity data fetching queries (GROQ). Instead, extract the fetched data and feed it as clean props into our newly designed high-fidelity interactive components.

## Target Aesthetic
- **"Deep Space Creative Tech"**: Zero-compromise premium dark theme. 
- **Palette**: Deep zinc and slate gray base backgrounds, layered glassmorphism cards, blurred neon ambient spots, and single-accent electric cyan (`cyan-500`) to highlight primary actionable elements and text highlights.
- **Physicality**: Introduce micro-textures like subtle static noise overlays and sharp, low-opacity borders to give the website a high-end tactile feel.
